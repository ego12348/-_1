## Висновок
Під час виконання першої частини лабораторної роботи №1 я:
* Ознайомився з інтерфейсом та структурою проектів у середовищі Code::Blocks.
* Вивчив базову структуру програми на мові С, зокрема роботу з головною функцією `main()`.
* Навчився використовувати функції `printf()` для виведення тексту та форматованих чисел (`float`) з різною кількістю знаків після коми.
* Отримав навички розрахунку середнього арифметичного значення для групи змінних.
* Навчився створювати репозиторії на GitHub, завантажувати туди вихідний код програми та оформлювати звіт у форматі Markdown.
